Снимок всего, что бот и веб-студия умеют **сегодня**: продукты, какие входные
параметры мы **сейчас передаём** в fal.ai, и какие параметры провайдера мы
**пока НЕ отдаём пользователю** (но можем — идеи для модификаций и промо).

Единица оплаты — **🔫 патрон** (1 патрон ≈ $0.02 себестоимости ИИ). Цена в
патронах = ceil(стоимость / 0.02); маржа на пакетах ≥3.5×.

### Входные параметры: что передаём vs что доступно (идеи модификаций)

| Модель | Передаём сейчас | НЕ отдаём (есть в fal docs) → идея |
|---|---|---|
| Seedream v4 (t2i) | `prompt` | `image_size`/aspect, `num_images`, `seed`, `negative_prompt` → **вертикаль 9:16 для TikTok**, вариации-сетка «4 в 1», сид для повторяемости |
| Nano Banana 2/Pro (t2i) | `prompt`, `resolution` (1K/2K) | `aspect_ratio`, `num_images`, `output_format` → **9:16 промо-кадры**, батч-вариации |
| Nano Banana edit | `prompt`, `image_urls` | несколько reference-картинок, `strength` → **смешать 2 фото** (человек+товар), контроль силы правки |
| GPT-Image-2 | `prompt`, `quality:high`, `image_size` | `background:transparent`, `n`, размеры/форматы → **PNG без фона для карточек**, серия |
| Kling 3.0 / 2.5 | `prompt`, `start_image_url`/`image_url`, `duration`, `aspect_ratio` | **`end_image_url` (кадр-конец)**, `cfg_scale`, `negative_prompt` → переход между двумя кадрами, сила следования промпту |
| Seedance 2.0 / Fast | `prompt`, `image_url`, `resolution:720p`, `duration`, `aspect_ratio` | **`1080p`** (флагман), **end-frame**, `4–15с` (сейчас 5/10), `seed`, синхронный звук-настройки → HD-тариф, длинные истории, референс-видео |

**Быстрые победы для промо-контента:**
1. **Aspect ratio на фото-моделях** — сейчас только у видео. Вертикаль 9:16 нужна
   для Reels/TikTok — легко добавить `aspect_ratio` в `input` t2i-моделей.
2. **`num_images` / вариации** — «4 варианта за 1 тап» = вау-контент для рекламы.
3. **Прозрачный фон (GPT-Image `background:transparent`)** — карточки товара PNG.
4. **End-frame у видео** — эффект «было → стало» одним роликом (сильный хук).
5. **1080p Seedance** — премиум-апселл для видео.

## 2. Пресеты (одним тапом, движок `nb2_edit`, 4 🔫)

**Фото (AI-фотосессия):** 💼 Бизнес-портрет · 🕶 Fashion · 🌅 Закат на Санторини · 🎥 Кино-портрет.
**Товар:** 🛍 Продающая карточка · ⬜️ Белый фон · 🌿 Lifestyle.

Каждый — курируемый промпт с сохранением личности (identity-lock). Вход:
`prompt` (курируемый) + `image_urls` (фото пользователя). Параметр для мода:
**aspect_ratio** (вертикаль для соцсетей), **num_images** (выбор из нескольких).

**Квиз-конструктор истории** (сказка, мини-фильм): выбор героя/спутника/финала/
эпохи/тона + «пара слов от себя». Фрагменты промпта склеиваются на сервере.

**Гарантии качества промптов:** реальный ребёнок — главный герой на переднем
плане в фокусе; ровно один экземпляр персонажа (без клонов); у звёзд — без
дублей фигур.

## 4. Видео-композер (веб-студия) — новое

Для **любого** фото→видео: формат кадра (**Как фото / 9:16 / 16:9 / 1:1**),
длительность (**5с / 10с** с живой ценой), сюжет (что происходит / жанр /
эмоция / камера) и персонализация (хобби, питомец, что любит). Цена видео
масштабируется по секундам. Параметры для мода: **end-frame**, **1080p**,
**4–15с** (Seedance), **negative_prompt**.

## 5. Витрина новинок (веб) + ежедневная рутина дешёвой модели

- **Баннер «🆕 Новые модели»** наверху студии: слайдер карточек, каждая — «Попробовать»
  прямо в студии. Самая дешёвая модель помечается **🎁 БЕСПЛАТНО** (входит в
  бесплатные патроны). Источник — `MODEL_NEWS` в `src/models.ts`.
- **Ежедневная строка в /dash и утренней сводке**: `cheapestModel()` считает
  самый дешёвый вход по типу и проверяет, покрывает ли его бесплатная проба —
  автоматически двигается при изменении цен fal.

## 8. Идеи промо под TikTok / Facebook (по продуктам)

- **Сказка/Мини-фильм** — самый сильный эмоциональный хук (родители). Ролик
  «фото ребёнка → сказка со звуком за 30 сек». Вертикаль 9:16.
- **Матч мечты** — сезонный, «выйди на поле с Месси». Быстрый вирал.
- **Старое фото** — «оживи бабушкино фото», плачущие комментарии = охваты.
- **Карточка товара** — для селлеров WB/Ozon, ROI-хук «фото за 40 сек вместо студии».
- **Видео-композер** — демо «одно фото → 5 разных роликов» (жанры/эмоции).
- Каждому креативу — свой deep-link `?start=src_<канал>` для per-source воронки в `/dash`.
