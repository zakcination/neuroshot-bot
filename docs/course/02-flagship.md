# $50 «AI-контент под ключ» — 3 модуля

*Флагман. Формат: 3 модуля + когорт-чат + сертификат по итогу, включено
500 🔫. Для тех, кто прошёл «Быстрый старт» (цена $9 засчитывается — см.

Здесь курс выходит за рамки одного тапа: вы всё ещё не пишете промпты
вручную для базовых результатов (для этого есть пресеты), но учитесь
**собирать из готовых блоков** то, что раньше требовало часов ручного
прогона в VN + отдельных нейросетей — персонажные серии, мини-фильмы со
звуком, и наконец бизнес поверх этого навыка.

## Модуль 1. Фото и аватары — техника identity lock

**Идея модуля.** Все "продвинутые" персонажные пайплайны в ручном режиме
(смена одежды с сохранением лица, разные ракурсы одного персонажа, старение/
омоложение) держатся на одном приёме — **жёсткой фиксации личности** между
кадрами. В боте это не отдельный навык, а свойство каждого пресета — но
чтобы использовать это осознанно, нужно понимать, как это работает.

### Пайплайн персонажа (4 шага)

Классическая последовательность — то же ТЗ, что мастер-группа даёт как
первое обязательное домашнее задание, только каждый шаг тут — один тап:

1. **Крупный план с лицом** → `/start` → 🖼 Редактирование фото → любой
   портретный пресет (**💼 Бизнес-портрет**, **🎥 Кино-портрет**)
2. **Дальний план того же человека**: в полный рост или по пояс, с одним
   предметом (сумка, телефон) → тот же пресет, другое фото или **Свой
   промпт** с явным описанием кадра
3. **Смена образа**: другая одежда, аксессуары → пресеты вроде **🕶
   Fashion-съёмка** или **📼 Плёнка 90-х** меняют стиль, сохраняя лицо
4. **Оживление**: → **🎬 Оживить фото** на лучшем результате

Во всех четырёх шагах работает один и тот же guard, встроенный в промпт
каждого пресета:

> Keep the person's face and identity exactly as in the photo.

Это то, что в ручном промпт-инжиниринге пишут явно каждый раз (и часто
забывают, отчего лицо "плывёт" между кадрами) — здесь это гарантировано
автоматически.

### Когда нужен «Свой промпт»

Для нестандартной задачи (не покрытой готовым пресетом) в боте есть режим
**«Свой промпт»** — вы описываете сцену текстом, а guard на identity
добавляется автоматически. Структура сильного промпта для сложной кадровой
сцены (кино-стилистика, разобранная в оригинальном курсе):

```
[эпоха/место] → [кто в кадре + эмоция] → [свет: контровой/боковой/золотой час]
→ [камера: план, уровень глаз, объектив 35мм] → [фон: размытый/резкий]
→ [стиль/движок: cinematic 3D / фотореализм] → [формат: 16:9, ультра-разрешение]
```

Пример (адаптация классического рабочего промпта из индустрии):

> 1990-е годы. Утро. Кухня. Тёплый свет мягко подсвечивает фигуры, создавая
> контровой свет. Камера: средний план, уровень глаз, объектив 35 мм. Фон
> слегка размытый. Cinematic 3D-анимация, реалистичный стиль, Pixar-уровень
> детализации. 16:9, ультра высокое разрешение.

Это ровно та же формула, что использует кампания **🎞 Мини-фильм с вами** —
загляните в её пресеты (**🌅 Тёплая драма**, **📼 Ретро 90-х**, **⚔️ Эпичное
кино**) как в готовый образец: собственный промпт полезно один раз собрать
руками именно для того, чтобы потом узнавать эту структуру в чужих
референсах.

**Задание модуля:** соберите персонажную серию из ваших 4 шагов, включая
минимум один "Свой промпт" по формуле выше.

## Модуль 2. Видео и оживление — сторибординг и озвучка

**Идея модуля.** Один ролик — это ремесло одного тапа. Серия роликов,
которая рассказывает историю (сериал, рекламная серия, кейс клиента) —
это сторибординг: список кадров, где под каждым — визуал + текст
рассказчика.

### Структура кадр + рассказчик

Каждый кадр серии описывается двумя строками:

```
Кадр N: [что видно на экране]
Рассказчик: [что говорит закадровый голос — 1-2 предложения]
```

Пример структуры (8 кадров, сериал-формат — от завязки к развязке):

| Кадр | Визуал | Рассказчик задаёт |
|---|---|---|
| 1 | Завязка — герой в начальной точке | Кто герой, где он сейчас |
| 2 | Первое препятствие | Проблема, с которой герой сталкивается |
| 3–4 | Попытки, труд | Нарастание сложности, стойкость героя |
| 5 | Поворотная точка | Момент решения/перемены |
| 6 | Первый результат | Маленькая победа |
| 7 | Развитие | Как результат меняет жизнь героя вокруг |
| 8 | Финал | Итог + текст на экране, закрепляющий смысл |

Каждый кадр в боте — это отдельная генерация через кампанию **🎞 Мини-фильм
с вами** (собственное фото + сцена из квиза "когда/тон") или через **🖼
Редактирование фото** для промежуточных состояний персонажа. Собираете
кадры по порядку в любом видеоредакторе на телефоне — бот делает контент,
монтаж кадров в серию остаётся вашим ремеслом (как и в оригинальном курсе:
навык монтажа не заменяется, а ускоряется).

### Озвучка

Бот не озвучивает текст рассказчика — это отдельный шаг двумя внешними
инструментами:
- **@steosvoice_bot** → голос "Старец рассказчик" — быстрый бесплатный
  вариант для СНГ-аудитории
- **ElevenLabs** (elevenlabs.io) → более широкий выбор голосов и языков,
  платно

Вставляете готовую дорожку в видеоредактор поверх смонтированной серии.

### Апгрейд: «со звуком» одним тапом

Для одного ролика (не серии) кампания **🎞 Мини-фильм с вами** уже включает
опцию **🎞 Снять мини-фильм (со звуком)** — рендерит многокадровую сцену с
собственным амбиентным звуком на флагманском видео-движке, без внешней
озвучки. Это самый дорогой, но и самый "готовый" формат в каталоге — до
76 🔫 за ролик, из выданных 500 🔫 модуля хватает на несколько таких попыток.

**Задание модуля:** соберите 6–8-кадровую серию по структуре выше, озвучьте
через steosvoice или ElevenLabs, смонтируйте в один ролик.

## Модуль 3. Продажи — от контента к клиентам

**Идея модуля.** Навык без дистрибуции не монетизируется. Этот модуль — как
превратить то, что вы научились делать в модулях 1–2, в первых клиентов и
первый доход.

### Три способа заработать на навыке

1. **Продавать карточки и ролики бизнесам** — тот же формат, что в Уроке 2
   курса «Быстрый старт», но как услугу: находите малый бизнес без нормального
   контента (маркетплейс-продавцы, локальные бренды), делаете карточку/ролик
   за оплату
2. **Вести собственный блог** — публикуете по формуле из Урока 5, растите
   аудиторию, монетизируете рекламой/своими услугами
3. **Партнёрская программа NeuroShot** — получаете свой код (`?start=c_<код>`),
   приводите аудиторию в бота, зарабатываете % с каждой их покупки —
   `/partner` показывает live-статистику (детали: `docs/creator-program.md`)

### 7-дневный плейбук запуска

Адаптация проверенной схемы прогрева (изначально — под запуск собственного
обучения; здесь — под запуск ваших AI-контент-услуг):

| День | Что публикуете |
|---|---|
| 1. История | «Ещё месяц назад я вообще не думал, что буду делать контент через ИИ» — честно: как узнали, почему начали, что было сложным |
| 2. Результат | Покажите свои лучшие карточки/ролики + метрики (просмотры, охват). Не продавайте — пусть люди сами спрашивают |
| 3. Возражения | «Почему у большинства не получается» — не в человеке дело, а в отсутствии системы (хаотичные попытки, нет пошагового пути) |
| 4. Польза | Бесплатная ценность: «5 промптов, которые я использую» или ссылка на бесплатный гайд ([`00-free-guide.md`](00-free-guide.md)) — доверие резко растёт |
| 5. Спрос | Опрос: «Если бы я предложил сделать вам такую карточку/ролик под ключ — было бы интересно?» Собираете реакции, не называя цену |
| 6. Анонс | Показываете программу услуги: что входит, для кого, без цены — только ценность |
| 7. Продажа | Открываете набор: для кого, что получат, сколько стоит, бонус первым, лимит мест («беру только первых N — чтобы успевать сделать качественно»), явный призыв к действию |

**Задание модуля (финальное задание всего курса):** пройдите 7-дневный
плейбук от начала до конца с реальной публикацией каждый день, приведите
минимум одного платящего клиента или партнёрского перехода. Разбор в
когорт-чате, сертификат — по итогу.
